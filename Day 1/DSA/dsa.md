# Problem: Find the Second Largest Number

## Description
Given an array of integers, find the second largest number.

## Input
An array of integers.

## Output
Return the second largest number.

## Example
Input: [10, 5, 8, 20]
Output: 10

## Constraints
- Do not use built-in sort directly for the final answer.
- Try to solve in one pass.

## Task
Write a function that returns the second largest element.

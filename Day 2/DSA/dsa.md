# Problem: Move All Zeros to End

## Description
Given an array, move all zeros to the end while maintaining the order of non-zero elements.

## Input
An array of integers

## Output
Modified array with zeros at the end

## Example
Input: [0, 1, 0, 3, 12]
Output: [1, 3, 12, 0, 0]

## Constraints
- Do it in-place (no extra array)
- Maintain order of non-zero elements

## Task
Write a function to solve this efficiently.

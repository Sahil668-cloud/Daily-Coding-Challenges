# Problem: Find First Non-Repeating Element

## Description
Given an array, find the first element that does not repeat.

## Input
An array of integers

## Output
First non-repeating element

## Example
Input: [4, 5, 1, 2, 0, 4]
Output: 5

## Constraints
- If all elements repeat, return -1

## Task
Solve efficiently without using nested loops if possible.
